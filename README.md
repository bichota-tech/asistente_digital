# Check! — Asistente Digital (learning in progress)

Proyecto frontend estático que simula un asistente digital con tres roles (estudiante, ciudadano, empresa).  
Su objetivo es practicar **lógica de tareas con JavaScript**, **persistencia local** y **UI responsiva**.

Static frontend project that simulates a digital assistant with three roles (student, citizen, company).  
Its goal is to practice **task logic in JavaScript**, **local persistence**, and **responsive UI**.

---

## 🎯 Propósito / Purpose (aprendizaje / learning)
Este proyecto me permite:

- Consolidar manejo de **localStorage**
- Practicar validaciones básicas con JavaScript
- Diseñar una UI adaptable a diferentes pantallas
- Organizar vistas por rol de usuario

This project helps me:

- Strengthen **localStorage** usage
- Practice basic JavaScript validations
- Design a UI that adapts to different screen sizes
- Organize views per user role

---

## ✨ Funcionalidades actuales / Current features
- Gestión de tareas (añadir, listar, eliminar)
- Validaciones: fecha no pasada y detección de duplicados
- UI responsiva: modal en móvil, panel en escritorio
- Persistencia local por rol usando `localStorage`

---

## 🧩 Tecnologías / Tech stack
- **HTML5**
- **CSS3**
- **JavaScript ES6**
- **localStorage API**

---

## 🧠 Aprendizajes clave / Key learnings
- Manejo de estado local sin backend
- Validaciones de datos en UI
- Separación de vistas por tipo de usuario
- Diseño adaptable en distintos breakpoints

---

## ⚠️ Limitaciones actuales / Current limitations
- Sin backend (no sincroniza tareas en la nube)
- Sin autenticación de usuarios
- Sin tests automatizados

## 🔮 Próximas mejoras / Next steps
